# Simple Redirect Website

This project is a simple web application that demonstrates how to create a button that redirects users to a specified link when clicked.

## Project Structure

```
simple-redirect-website
├── index.html       # Main HTML document
├── styles.css       # CSS styles for the website
├── script.js        # JavaScript code for handling button click
└── README.md        # Documentation for the project
```

## Getting Started

To set up and run the website, follow these steps:

1. **Clone the repository** (if applicable):
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd simple-redirect-website
   ```

3. **Open the `index.html` file** in your web browser:
   You can simply double-click the `index.html` file or open it using your preferred web browser.

## Usage

- Click the button on the webpage to be redirected to the specified link.

## Customization

- You can modify the link in the `script.js` file to change the redirection target.
- Adjust the styles in `styles.css` to customize the appearance of the button and other elements.
